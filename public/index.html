Tovias Publishing Toolkit

This project is a Python/Flask application. Netlify hosts static sites; it cannot run the Python server directly.

Run locally

python -m venv .venv
source .venv/Scripts/activate
pip install -r requirements.txt
python main.py grammarly
# Open http://127.0.0.1:5000/designer

Deploy with Docker

docker build -t tovias-publishing .
docker run -p 8000:8000 --env-file .env tovias-publishing
# Open http://localhost:8000/designer

Or deploy on a Python-friendly host (Render, Railway, Fly.io, Azure Web Apps, etc.)

Use the provided Dockerfile or Procfile.
